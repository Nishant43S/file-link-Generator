File Link Generator

File Link Generator is a modern web application that allows users to effortlessly upload files and generate secure, temporary download links. Developed using Flask and Python as the backend framework, it integrates Dropbox for cloud file storage and Firebase for dynamic data handling and optional user tracking.

The frontend is crafted using HTML, CSS, and JavaScript, providing a clean and responsive user interface with features like drag-and-drop file upload, real-time progress display, and QR code generation for download links. Once a file is uploaded, the app creates a temporary download link that auto-expires after a set time (e.g., 30 minutes), ensuring both security and efficiency. The file is also programmatically deleted from Dropbox after the link expires to maintain storage hygiene.

This project demonstrates expertise in integrating cloud services, secure file handling, and creating user-friendly web interfaces. It serves as a practical and innovative solution for safe and time-bound file sharing.
