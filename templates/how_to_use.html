How to Use the App

Step 1: Register or Login

First, you need to create an account. You can do this by signing up using your email and a password. If you already have an account, simply log in using your registered email and password.

Step 2: Reset Password (If Needed)

If you forget your password, don't worry! Just click on the "Forgot Password?" option on the login page. Enter your email address, and a reset link will be sent to your inbox. Follow the link to set a new password.

Step 3: Enter Access Token

After logging in, you'll be taken to the Access Token page. Here, you need to paste the **Dropbox Access Token** to activate the app's file upload feature. This token connects the app to your Dropbox for secure file storage.

Step 4: Upload Your File

Once the access token is verified, the file uploader page will appear. You can now upload any type of file, such as:

  • Images (.png, .jpg, .jpeg, etc.)
  • Documents (.pdf, .docx, etc.)
  • Compressed files (.zip, .rar)
  • Any other file types

Just drag and drop or click to select the file. After uploading, the app will generate a **secure, temporary download link**. You can also scan the **QR code** to download the file directly on your phone.

The link will expire automatically after 30 minutes, and the file will be deleted from Dropbox for your safety.

That's it!

Enjoy secure, easy, and temporary file sharing with the File Link Generator!
